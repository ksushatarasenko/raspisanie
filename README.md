# School Planner — Дневник с задачами (статический)


**Что делает этот проект**
- Красочное школьное расписание (понедельник — пятница), 7 уроков в день
- Навигация по неделям (влево/вправо) с указанием дат
- Если в расписании нет изменений — расписание обновляется автоматически для новой недели
- Возможность редактировать уроки (предмет, время, кабинет) и сохранять изменения
- Секция задач под расписанием: добавить задачу (название, время, место) — появится под расписанием
- Задачи сохраняются в `localStorage`; есть экспорт/импорт JSON и очистка данных
- Каждая задача можно пометить как выполненная/не выполненная


**Файлы**
- `index.html` — главная страница
- `styles.css` — стили
- `app.js` — логика: управление неделями, редактирование расписания, задачи, сохранение


**Как запустить**
1. Клонируйте репозиторий
2. Поместите файлы в корень
3. Откройте `index.html` в браузере или разверните через GitHub Pages


**Для продакшн‑сохранения**
- Рекомендую подключить сервер‑бэкенд или BaaS (Firebase / Supabase) вместо `localStorage` — тогда данные будут храниться в облаке и синхронизироваться между устройствами.
